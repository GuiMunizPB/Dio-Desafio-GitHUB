INTRODUÇÃO AO GIT

WINDONS - PROMPT
COMANDO DIR : LISTA AS PASTAS DO DIRETÓRIO
COMANDO CD : NAVEGAÇÃO ENTRE AS PASTAS
COMANDO CD / : / PARA LOCAL ESPECÍFICO DA NAVEGAÇÃO
COMANDO CD .. : RETORNAR 1 PASSO
CLS : LIMPA TELA
MKDIR WORKSPACE : CRIAR PASTAS
ECHO HELLO > HELLO.TXT : CRIANDO ARQUIVOS
DEL : COMANDO DE DELETAR
RMDIR WORKSPACE /S /Q : DELETANDO ARQUIVOS

PASSAR O SHA1 NO GIT:
echo 'EXEMPLO' | git hash-object --stdin
OBJETOS FUNDAMENTAIS (BLOOBS) - ARMAZENAR OBJETOS COM METADADOS
OBJETOS FUNDAMENTAIS (TREE) - ARMAZENAR BLOOBS COM O NOME DO ARQUIVO E POSSUEM SHA1 PRÓPRIO
OBJETO MAIS IMPORTANTE (COMMIT) - APONTA PARA TREE, PARENTE, AUTOR, MENSAGEM E O TIMESTAMP E POSSUI SHA1 PRÓPRIO
